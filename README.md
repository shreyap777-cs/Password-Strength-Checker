# Password-Strength-Checker
A simple and interactive Password Strength Checker Web Application built using Python and Streamlit. This project helps users analyze password security and improve weak passwords using smart suggestions.

Live Demo:
https://password-strength-checker-6yagzx3bxqvslgz3ypdzyr.streamlit.app/

Features:
1.Password Strength Score (0–100)
2.Suggestions to improve weak passwords
3. Password Entropy Calculation (randomness check)
4.Breach Check using Have I Been Pwned API
5.Common Pattern Detection (qwerty, 1234, repeated characters, common passwords)
6.Strong Password Generator with custom options

Tech Stack:
Python ,
Streamlit ,
Regex (re) ,
requests ,
hashlib ,
math ,
random

Files Included:
Password.py → Main Streamlit application ,
requirements.txt → Required dependencies ,
