# Password-Strength-Checker
A simple and interactive Password Strength Checker Web Application built using Python and Streamlit. This project helps users analyze password security and improve weak passwords using smart suggestions.

Live Demo:<br>
https://password-strength-checker-6yagzx3bxqvslgz3ypdzyr.streamlit.app/

Features:<br>
1.Password Strength Score (0–100) <br>
2.Suggestions to improve weak passwords <br>
3. Password Entropy Calculation (randomness check) <br>
4.Breach Check using Have I Been Pwned API <br>
5.Common Pattern Detection (qwerty, 1234, repeated characters, common passwords) <br>
6.Strong Password Generator with custom options <br>

Tech Stack:
Python ,
Streamlit ,
Regex (re) ,
requests ,
hashlib ,
math ,
random

Files Included:<br>
